# LayarFilm - Netflix Clone

A high-fidelity Netflix clone built with Vanilla HTML, CSS, and JavaScript.

## 🚀 Features

- **Dark Mode**: Original Netflix dark aesthetics.
- **Dynamic Content**: Movie cards are generated from a JSON-like data structure in `main.js`.
- **Responsive Design**: Works on Desktop and Mobile.
- **Premium Animations**: Smooth scaling and hover effects on movie cards.
- **Auto-Sync**: Automatically pushes changes to GitHub for GitHub Pages deployment.

## 📁 Structure

- `index.html`: Main page structure.
- `css/style.css`: Design system and styling.
- `js/main.js`: Core logic and data handling.
- `assets/`: Static media files.
